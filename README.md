# WP Schema Generator Plugin

## Description
This plugin allows users to generate and manage schema data inside WordPress easily.

## Features
- Add schema to pages/posts
- Detect missing or broken page URLs
- One-click schema generation
- Import and export schema data
- Generate PHP code snippet for reuse or backup

## Future Improvements
- Advanced schema templates
- Auto SEO structured data integration
- UI improvements for better usability
